# pkg/api - REST API Server

This package implements the REST API server for lakeFS, providing HTTP endpoints for all lakeFS operations.

## Overview

The API package provides:
- **REST API**: HTTP endpoints for repository, branch, commit, object operations
- **Authentication**: Multiple auth methods (JWT, Basic, Cookie, OIDC, SAML)
- **Authorization**: Permission-based access control
- **OpenAPI/Swagger**: Auto-generated from swagger.yml
- **S3 Gateway**: S3-compatible API (separate package)

## Authentication (`auth_middleware.go`)

### Supported Methods

The API supports multiple authentication methods defined in swagger.yml:

| Method | Type | Header/Cookie |
|--------|------|---------------|
| jwt_token | Bearer | Authorization: Bearer \<token\> |
| basic_auth | Basic | Authorization: Basic \<base64\> |
| cookie_auth | API Key | Cookie: internal_auth_session |
| oidc_auth | API Key | Cookie: oidc_auth_session |
| saml_auth | API Key | Cookie: saml_auth_session |

### Middleware Flow

```go
func AuthMiddleware(...) func(next http.Handler) http.Handler {
    // 1. Check if user already authenticated
    if user, _ := auth.GetUser(r.Context()); user != nil {
        return next // Skip auth
    }

    // 2. Extract security requirements from OpenAPI spec
    securityRequirements, _ := extractSecurityRequirements(router, r)

    // 3. Check each security requirement
    user, err := checkSecurityRequirements(r, securityRequirements, ...)

    // 4. Add user to context
    ctx := logging.AddFields(r.Context(), logging.Fields{logging.UserFieldKey: user.Username})
    r = r.WithContext(auth.WithUser(ctx, user))

    return next
}
```

## OpenAPI/Swagger Integration

### swagger.yml (`api/swagger.yml`)

The OpenAPI 3.0 specification defining all endpoints:

```yaml
openapi: "3.0.0"
info:
  title: lakeFS API
  version: 1.0.0

servers:
  - url: "/api/v1"

security:
  - jwt_token: []
  - basic_auth: []
  - cookie_auth: []
  - oidc_auth: []
  - saml_auth: []
```

### Generated Code (`apigen/lakefs.gen.go`)

Auto-generated by oapi-codegen from swagger.yml:

- **Types**: Go structs matching API schemas (e.g., `Commit`, `BranchCreation`)
- **Handler**: HTTP handler interface
- **Client**: Go client for the API

### Validation (`serve.go`)

The `OapiRequestValidatorWithOptions` middleware validates:
- Request parameters against OpenAPI spec
- Request body (except file uploads)
- Security requirements

## Key Endpoints

### Repositories
- `POST /repositories` - Create repository
- `GET /repositories` - List repositories
- `GET /repositories/{repository}` - Get repository
- `DELETE /repositories/{repository}` - Delete repository

### Branches
- `POST /repositories/{repository}/branches` - Create branch
- `GET /repositories/{repository}/branches` - List branches
- `GET /repositories/{repository}/branches/{branch}` - Get branch
- `DELETE /repositories/{repository}/branches/{branch}` - Delete branch

### Commits
- `POST /repositories/{repository}/branches/{branch}/commits` - Create commit
- `GET /repositories/{repository}/commits/{commit_id}` - Get commit
- `GET /repositories/{repository}/refs/{ref}/log` - Commit log

### Objects
- `GET /repositories/{repository}/refs/{ref}/objects` - List objects
- `GET /repositories/{repository}/refs/{ref}/objects/{path}` - Get object
- `POST /repositories/{repository}/branches/{branch}/objects` - Upload object
- `DELETE /repositories/{repository}/branches/{branch}/objects/{path}` - Delete object

### Tags
- `POST /repositories/{repository}/tags` - Create tag
- `GET /repositories/{repository}/tags` - List tags
- `DELETE /repositories/{repository}/tags/{tag}` - Delete tag

### Multipart
- `POST /repositories/{repository}/branches/{branch}/multipart` - Create multipart upload
- `POST /repositories/{repository}/branches/{branch}/multipart/{upload_id}/parts` - Upload part
- `POST /repositories/{repository}/branches/{branch}/multipart/{upload_id}/complete` - Complete

### Pre-signed URLs
- `GET /repositories/{repository}/refs/{ref}/objects/{path}/url` - Get pre-signed URL

## Common Tasks

### Adding a new endpoint:
1. Add to swagger.yml with proper schema
2. Regenerate apigen: `make gen-code`
3. Implement handler method in controller.go
4. Add authorization check

### Authentication flow:
1. Request comes in with auth header/cookie
2. AuthMiddleware extracts security requirements from OpenAPI spec
3. checkSecurityRequirements tries each method in order
4. First successful auth wins, user added to context

### Error handling:
```go
func (c *Controller) SomeOperation(...) {
    err := doSomething()
    if errors.Is(err, someErr) {
        writeError(w, r, http.StatusNotFound, "not found")
        return
    }
    if c.handleAPIError(ctx, w, r, err) {
        return  // Error was handled
    }
    writeResponse(w, r, http.StatusOK, result)
}
```
